# Planned Components

## Navbar

Primary navigation for Home, Solver, Play, How It Works, and About.

## Layout

Shared page shell for spacing, route outlet placement, and future theme handling.

## SudokuBoard

Interactive 9x9 board container responsible for rendering cells and coordinating board-level events.

## SudokuCell

Single-cell UI for givens, user-entered values, notes, selected state, related-cell highlighting, and validation state.

## ControlPanel

Action area for commands such as solve, clear, reset, hint, pause, and notes mode.

## DifficultySelector

Difficulty picker for Easy, Medium, Hard, and Expert puzzles.

## Timer

Elapsed-time display for Play mode.

## StatsPanel

Compact status display for difficulty, mistakes, hints, and game progress.

## MessageBox

Reusable feedback surface for validation messages, solver results, and gameplay updates.
